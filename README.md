# MP3 Tag Reader

## Project Overview
This project implements an MP3 Tag Reader using C programming. 
The program extracts metadata information from MP3 files by reading ID3 tags.

It displays details such as song title, artist name, album name, year of release, and genre.

## Features
- Reads ID3 metadata from MP3 files
- Displays song title, artist, album, year, and genre
- Demonstrates binary file handling in C
- Simple command-line interface

## Technologies Used
- C Programming
- File Handling
- Linux Environment

## How It Works
1. The program opens the MP3 file.
2. It reads the ID3 tag frames.
3. Extracts metadata information.
4. Displays the song details to the user.
